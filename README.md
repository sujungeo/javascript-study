# Todo List

## Overview

Vanilla JavaScript로 구현한 할 일 관리 애플리케이션

## Features

- Todo 항목은 to do, doing, done 라벨로 분류한다.
- 리스트를 보여줄 수 있는 Board 뷰가 있다.
- Todo 항목을 추가할 수 있다.
- Todo 항목을 삭제할 수 있다.

## Getting Started

1. 프로젝트 clone
2. `npm install`
3. `npm start`

## Dependencies

- Node.js

### DevDependencies

- Webpack
